# Retail-Customer-Sales-Analytics-Dashboard
Interactive Power BI dashboard project analysing retail sales performance, customer behaviour, and inventory demand trends using Business Intelligence and data visualisation techniques.

## Project Overview
This project focuses on transforming retail transaction data into actionable business insights through interactive dashboards designed for Sales, Marketing, and Procurement stakeholders. The dashboards support data-driven decision-making by identifying customer trends, sales performance patterns, regional demand differences, and operational opportunities.

The project demonstrates the practical application of Business Intelligence, KPI reporting, customer analytics, and data visualisation techniques to solve real-world business problems.

# Tools & Technologies
- Power BI
- SQL
- Excel
- Data Visualisation
- KPI Reporting
- Business Intelligence Analytics
- Customer Behaviour Analysis

# Dataset
The project utilised a retail supermarket sales dataset containing 1,000+ transaction records across:
- Product categories
- Customer segments
- Payment methods
- Cities and regions
- Monthly and daily transactions
- Customer ratings and purchase behaviour
